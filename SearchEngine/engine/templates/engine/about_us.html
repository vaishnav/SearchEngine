How Our Search Engine works

There are multiple steps involved, from user entering the query, the processing of query in the backend, providing best result for the query and Giving an intuitive and friendly UI to the whole process.

Ranking

Ranking is a process where we figure out quality of our webpages. We are using a variant of Page Rank Algorithm to rank webpages from best to worst. Page Rank was originally developed by Larry Page (Co-Founder of Google) and Google still uses it. For our case we are using backlinks (number of other pages that have links to our current page) as a parameter for judging. The current variant of PageRank we are using is know as random surfer model.

Important points

  • We first figure out how sites are linking to one-other
  • points regarding random surfer model
  • Ranking process gives us a rank about how well our pages holds compare to other pages on network

Indexing

Indexing is the process where we process all the data present on a page. We are using Aritificial Interlligence and Natural Language Processing to filter out the content of page from unnecessary terms and make a dataset of all the key terms of a webpage. We are using Cosine Similarity to check how which webpage best match to the given query.

Important points

  • Filter out webpages using AI and natural language processing
  • Using Cosine Similarity to determine which page best suits for given query

Crawling

Crawling is the process where we got to each page one by one and collect necessary infomation from them. We implement a basic web crawler, which is a bot that starts from a predefined web-page and traverse through all the pages one by one and collectign information like linking of sites and this information is transferred to Ranker and Indexer for later use

Important points

  • Designing a web-crawler that starts from predefined page and traverse to each page
  • While traversing each site necessary data is collected
